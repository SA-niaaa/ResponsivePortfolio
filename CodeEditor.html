Code Editor

A lightweight and functional web-based code editor created with HTML, CSS, and JavaScript that enables users to write, edit, and preview HTML, CSS, and JavaScript code in real time. The editor features separate panels for each language and an integrated live preview window, allowing immediate visualization of the output as users type. It incorporates iframe rendering for dynamic output, event listeners for real-time updates, and a responsive interface to ensure usability on both desktop and mobile devices. Designed as a practical tool for beginners and enthusiasts, it highlights core web development concepts such as, DOM events, and cross-browser compatibility. This project reflects the ability to create interactive web tools while maintaining simplicity and performance.
