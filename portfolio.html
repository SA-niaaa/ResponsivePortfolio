Responsive Portfolio

A personal portfolio website developed using HTML, CSS, and JavaScript to showcase projects, skills, and contact information. The website is fully responsive, ensuring it works well on both desktop and mobile devices. It includes sections like About, Skills, Projects, and Contact, presented with a clean layout and easy navigation. This project demonstrates the use of front-end technologies and responsive design principles to create a functional and professional online presence.
